A simple API that serves meal planning data from Google Calendar as JSON.

API Endpoint

GET /meals.json

Response Format

The API returns a JSON object with day indices (0-6) as keys:

  • 0 = Monday
  • 1 = Tuesday
  • 2 = Wednesday
  • 3 = Thursday
  • 4 = Friday
  • 5 = Saturday
  • 6 = Sunday

Example Response

{
  "0": {
    "Breakfast": "",
    "KidLunch": "SunButter Energy Bites",
    "Lunch": "Crispy Cabbage Pancakes",
    "Dinner": "Chicken Tikka Masala"
  },
  "1": {
    "Breakfast": "",
    "KidLunch": "Mama French Toast",
    "Lunch": "Crispy Cabbage Pancakes",
    "Dinner": "Chicken Tikka Masala"
  }
}

Usage

Once deployed, your API will be available at:

https://yourusername.github.io/meal-plan-api/meals.json

Meal Times

  • Breakfast: 8 AM
  • KidLunch: 11 AM
  • Lunch: 12 PM
  • Dinner: 5 PM
